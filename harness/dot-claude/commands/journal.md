# CE 사고 여정 생성

현재 작업의 CE(Context Engineering) 사고 여정을 `.thoughts/` 폴더에 기록합니다.

## 기록 항목

1. **컨텍스트 수집 (Gather)** — 어떤 파일을 읽었고 왜
2. **정보 선택/폐기 (Select)** — 채택/폐기한 정보와 이유
3. **실패 모드 감지 (Detect)** — Poisoning/Distraction/Confusion/Clash
4. **대안 비교 및 결정 (Decide)** — 대안 표 + 결정 근거
5. **적용된 CE 전략** — Write/Select/Compress/Isolate
6. **핵심 통찰** — 재사용 가능한 CE 교훈

## 파일 형식

`.thoughts/YYYY-MM-DD-{subject}.md`
