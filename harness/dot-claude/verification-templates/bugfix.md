- [ ] 수정 후 재현 테스트 통과 — verify-cmd: `<재현 테스트>` exit 0

- [ ] 동일 클래스 결함 탐색 — 같은 패턴의 잠재 지점 grep/검사 결과 보고 (수정은 범위 밖 — 등재만)
