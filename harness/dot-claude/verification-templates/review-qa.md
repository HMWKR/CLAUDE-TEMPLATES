# DoD 템플릿: 검수·QA

- [ ] **카나리 캘리브레이션** — 검수 대상에 사전 심은 결함 `<N>`개 중 탐지 `<M>`개, 탐지율 ≥ 2/3. **미달 시 이하 "발견 0" 판정 전부 무효**(눈먼 수렴 방지)
- [ ] 렌즈 커버리지 — correctness / security / perf / a11y·UX 중 해당 렌즈 전부 실행, 렌즈별 결과 기록
- [ ] 판정단 독립성 — 검수자에 이종/상위 모델 ≥1 혼합, 만장일치 아닌 발견은 불확실 등급 보존
- [ ] 발견 전수 등재 — 등급 강등 없음(rules/uncompromising-rigor §2), 형식: 발견 #N/분류/위치/증거/재현
- [ ] 실행 증거 — 정적 읽기만으로 PASS 처리한 항목 0 (도구 실행 결과 첨부)
- [ ] 종결 조건 — 신규 발견 0 × 2회 연속(카나리 유효 전제) AND 잔여 미해결 High/Medium 0

전항 PASS/N/A 가 검수 종결 조건. 이 템플릿 자체를 verify-lock으로 잠근다.
