# DoD 템플릿: 리팩토링

- [ ] 사전 스냅샷 — 리팩토링 **전** 전체 테스트 통과 확인(증거: 실행 출력) — verify-cmd: `<전체 테스트>` exit 0
- [ ] 사후 동일 통과 — 동일 명령 exit 0, 테스트 파일 무변경(verify-lock으로 잠금)
- [ ] 동작 불변 증거 — 공개 인터페이스·출력 대조(스냅샷 diff 또는 특성 테스트)
- [ ] 범위 준수 — 요청된 대상만 변경. "겸사겸사" 개선 0 (발견한 개선 후보는 등재만)
- [ ] 삭제 코드 확인 — 제거된 심볼의 참조 0 확인 — verify-cmd: `<참조 검색 명령>` 결과 0건
- [ ] 성능 회귀 없음(해당 시) — 측정 수치 전후 대조

전항 PASS/N/A + verify-lock check OK 가 완료 조건.
