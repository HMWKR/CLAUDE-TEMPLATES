# [L0 어댑터] 모델 프로파일: 미검출 안전 기본값 (고처방)

> adapter-inject 자동 주입 — 세션 모델을 감지하지 못했거나 미지 모델(haiku 포함)임.
> 안전 원칙: 저처방을 하위 모델에 잘못 적용하는 것이 최악의 오탐이므로, 미검출은 항상 고처방으로 떨어진다(청사진 v2 공격 H 대응).

- **처방 수준: 고** — 착수 전 명시적 단계 목록으로 분해, 각 단계에 검증 체크, 순서 실행.
- **L1 증폭 N**: sonnet 프로파일과 동일(설계 5 / 구현 3 / 리뷰 4×3 / 디버그 5 / 모호성 5).
- **분해 입도**: fine, maxTurnSpan: short, 턴 경계 상태 재주입.
- **검증 주권**: DoD·verify-cmd 자기 저작 금지 — 템플릿 + verify-lock 고정.
- **발산 차단**: 동일 실패 3회 연속 → 중단·에스컬레이션 보고.
- **[Fable-파리티 규범] 출력 계약·자기검증·엣지**: ① 요구된 출력 형식만(부가 보고 금지) ② 산출 전 실행 검증 기본 ③ 엣지 점검(빈 입력·경계·방향·중복).
