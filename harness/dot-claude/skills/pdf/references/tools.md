# PDF Tools Reference

## Library Selection Guide
| Task | Library | Notes |
|------|---------|-------|
| Read text | pdfplumber | Best for tables |
| Merge/split | pypdf | Lightweight |
| Create new | reportlab | Full control |
| Fill forms | pypdf / pdf-lib | See forms.md |
| OCR scanned | pytesseract | Requires Tesseract |
